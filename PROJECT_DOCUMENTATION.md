# Blog Project Documentation


## Introduction

The Blog Project is a web application developed to create and manage blog content.


## Objective

The objective was to build a simple blogging platform while learning frontend development, backend APIs, Git, and deployment.


## Development Process

The project development included:

- Project setup
- Frontend development
- Backend API creation
- GitHub integration
- Deployment
- Testing
- Optimization


## Features


### Add Blog

Users can add blog posts using the blog form.


### View Blogs

Users can view available blog posts.


### Edit/Delete Blogs

Users can modify or remove blog posts.


## Testing

The application was tested for:

- Responsiveness
- Functionality
- User input validation
- API requests


## Conclusion

This project improved my understanding of full-stack web development and deployment workflows.
