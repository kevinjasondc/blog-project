# Blog Project

## Project Overview

A simple blog website built using HTML, CSS, JavaScript, and Node.js with Express. 
The application allows users to add and manage blog posts through an interactive interface.

## Features

- Responsive blog website design
- Add new blog posts
- View blog posts
- Edit blog posts
- Delete blog posts
- Express.js backend APIs
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Git & GitHub

## How to Run Project Locally
### 1. Clone Repository
git clone https://github.com/kevinjasondc/blog-project.git
### 2. Install Dependencies
npm install
### 3. Start Server
node server.js
### 4. Open Website
http://localhost:3000
